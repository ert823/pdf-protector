## 这软件是干什么的

PDF Protector 是一款专为 macOS 打造的 PDF 加密解密工具，来自德国独立开发商 Witt Software。开发者的思路很清晰：只做这一件事，把它做好。

它能做的事情主要有两个：

**1. 给 PDF 加密**
- 设置「打开密码」：不知道密码的人直接无法打开文件
- 设置「权限密码」：可以限制对方复制文字、打印内容、修改文件

**2. 去除 PDF 的密码保护**
- 把别人发来的加密 PDF、自己之前加过密的文件，解除密码保护，存成无密码版本

加密用的是标准的 Adobe PDF 加密协议，40-bit 和 128-bit 两种强度都支持。加密后的文件可以在绝大部分 PDF 阅读器里正常使用——不只是 PDF Protector，Preview、Adobe Reader、其他第三方 PDF 软件都没问题。

## 用起来是什么感觉

说"简单"有时候显得很空洞，这里描述一下实际操作流程：

打开软件，把需要加密的 PDF 文件**直接拖进窗口**。然后你会看到两个密码输入框——一个是打开密码，一个是权限密码，按需填写。最后选"保存为新文件"（这样原文件不动），确认一下，完成。

整个流程三步，没有多余的界面，没有什么需要学习的东西。

解密也一样：把加密的 PDF 拖进去，输入密码，软件把密码去掉，重新保存一份无密码版本。

如果你经常需要给一批文件加同一个密码，还可以设置**默认密码**，省去每次重复输入的麻烦。
